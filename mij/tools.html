Tools for Social Media Data Mining

Tools for social media analytics, retrieve messages etc.

  • Shared Count - input a link and get the number of times it has been shared, liked och commented on Facebook, Google+, Pinterest, LinkedIn and StumbleUpon.
  • BuzzSumo - search for content published on Facebook and Twitter.
  • The Digital Methods Initiative - tools for gathering research data from Twitter, Facebook, Google etc. Source code available.

Tools for Managing Data

Like Excel.

  • Google Fusion Tables.

Tools for Data Visualization

  • Tableau. Easy to use software for visualization.
  • ggplot in R. Visualizations for data science. Requires programming knowledge in R.
  • D3.js. Create visualizations for the web. Requires programming knowledge in JavaScript.

Tools for Programming

Programming Languages

  • Python and Anaconda, which contains Jupyter Notebooks.
  • R and code editor RStudio.

Code Editors

  • Sublime Text. Lightweight, works on Windows, Mac and Linux.
  • Visual Studio Code. Lightweight, works on Windows, Mac and Linux.
  • Visual Studio Community. Heavy, works on Windows and Mac.

Big Data

  • Google Big Query. Perform analysis on big data with Googles services.
  • Apache Spark. Cluster computing (will work on any computer as well). Requires that you install it on your own server. Useful to run with Jupyter Notebooks. A hosted alternative is provided by Databricks, among others.

Swedish Tools Only

  • Twittercensus - svenska Twitter.
  • Sociala Nyheter - nyheter delade i Sverige (dock felaktig delningsstatistik).
  • Datajournalistik.se - lista med smarta verktyg för datajournalistik.
